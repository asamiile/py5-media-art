# Coral Reef

A procedural coral reef simulation featuring organic marine structures and underwater atmosphere.

## Theme

Marine biology, underwater ecosystems, organic growth, biodiversity.

## Technique

Procedural coral generation with recursive branching, randomized parameters, depth-based layering, and underwater lighting effects.

## Visual Elements

- **Coral Types**: Five distinct coral morphologies (branching, plate, staghorn, brain, fan)
- **Color Palette**: Deep ocean navy, vibrant coral pink, seafoam teal, golden amber, soft lavender, bright turquoise
- **Atmosphere**: Underwater light rays, rising bubbles, depth-graded sea floor
- **Organic Variation**: Randomized branching angles, length decay, and colony placement

## Parameters

- 8-12 coral colonies per render
- 3-6 branches per colony
- 3-7 recursion depth depending on coral type
- 5-8 light rays
- 20-30 bubbles

## Color Scheme

- Deep Navy: (10, 15, 35) - Background and sea floor
- Coral Pink: (255, 127, 127) - Branching coral
- Seafoam Teal: (64, 224, 208) - Plate coral
- Golden Amber: (255, 193, 7) - Staghorn coral
- Soft Lavender: (200, 180, 220) - Brain coral
- Bright Turquoise: (0, 206, 209) - Fan coral
- Ocean Blue: (30, 60, 100) - Depth gradient

## Notes

Each render produces a unique coral reef configuration with organic variation in branching patterns, colony placement, and atmospheric effects. The recursive branching algorithm creates naturalistic coral structures while maintaining visual coherence.
