# signal_fossil

A damaged future broadcast appears as a luminous fossil embedded in a dark substrate.

## Concept

The image treats a signal as something physical: compressed, fractured, partially erased, and slowly excavated from a black technological sediment.

## Technique

- **Gabor-like Wave Packet Synthesis**: Localized directional wave fields overlap into a broken carrier signal.
- **Contour Quantization**: Continuous wave energy is sliced into fine contour bands, turning smooth interference into sharp luminous strata.
- **Packet Dropout Masking**: A tile-based dropout field erases sections of the signal, creating modern compression artifacts without using a literal screen motif.
- **Additive Multi-Hue Glow**: Cyan, amber, violet, and rose passes separate active edges, missing data, and high-energy reconstruction ticks.

## Visual Impression

A full-canvas abstract relic of light: fractured cyan lines, amber stress edges, violet missing-data shadows, and vertical recovery ticks glowing from a near-black field.
