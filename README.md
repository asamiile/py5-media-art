## Media Art Works

- Install ffmpeg (once)

```bash
brew install ffmpeg
```

- Run Sketch

```
uv run python sketch/test/main.py
```

- Autonomous artwork creation via Claude Code

Open this repository in Claude Code and run:

```
/create-artwork
```

If you continue with multiple iterations:

```
/create-artworks
```
