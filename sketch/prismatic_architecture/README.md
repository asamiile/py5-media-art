# prismatic_architecture

A futuristic, modular city built from shifting crystalline structures that refract and shimmer.

## Concept

This artwork imagines an architecture grown rather than built, using recursive geometric principles. It simulates a world of obsidian skies and iridescent habitats, where every surface is coated in a thin-film material that diffracts light into a spectrum of colors.

## Technique

- **Recursive Isometric Subdivision**: Clusters are formed by subdividing space into smaller isometric cubes, creating a hierarchical, fractal-like structure.
- **Thin-Film Interference Simulation**: The color of each face is determined by a simulated thickness parameter, mapped to a periodic spectral function. This creates the "oil-on-water" or "iridescent" effect.
- **Additive Overlap**: Using the `SCREEN` blend mode, the overlapping transparent crystals create high-luminosity "hotspots" where structures converge, simulating intense internal reflections.
- **Organic Growth Patterns**: Random jitter and conditional subdivision probabilities ensure that each cluster has a unique, non-uniform silhouette.

## Visual Impression

A cluster of glowing, geometric crystals that seem to change color as the eye moves across them; high-contrast rainbow flares on the edges and deep, saturated interiors against a void of obsidian.
