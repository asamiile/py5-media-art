# neural_synapse

A generative artwork simulating the electrical activity of a biological neural network.

## Concept
The piece explores the complex, flickering beauty of neural connectivity. It visualizes the "action potentials" (electrical signals) that race through the brain's dense web of dendrites and axons, connecting soma to soma in a constant dance of information.

## Technique
- **Stochastic Branching Growth**: Neural paths are generated using a random-walk branching algorithm starting from several "somas" (neuron centers).
- **Network Path Rendering**: The base network is rendered with semi-transparent, tapered lines to give it a biological, fibrous feel.
- **Signal Pulse Simulation**: Over 1,000 "action potential" pulses are simulated by drawing multi-scale glowing segments along the network paths.
- **Multi-Scale Glow**: Signals use overlapping strokes of varying widths and opacities to create a convincing "electrical pulse" effect.
- **Organic Somas**: Neuron centers are rendered as irregular, glowing polygons to mimic the non-geometric nature of biological cells.
